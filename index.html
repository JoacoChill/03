1.1-¿Qué significa CSS?

CSS significa Hojas de Estilo en Cascada en inglés Cascading Style Sheets y te dará los estilos (diseño) necesarios para que un sitio web o una aplicación web se vean visualmente bien.

1.2-¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

El CSS es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc.

1.3-¿A qué se llama regla de CSS?

Las reglas de CSS son un conjunto de propiedades y valores que se aplicarán a una o varias etiquetas según necesitemos.

1.4-¿Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

La anatomia de la regla CSS se con forma por: primero el selector y luego una declaracion cumpuesta por una propiedad y el valor de la propiedad.

REGLA DE CSS

Selector
selecciona el(los) elemento(s) a dar estilo.
declaracion
aplica las propiedades que querés dar al elemento seleccionado en el Selector.
propiedad
es lo que te permite dar un estilo a un elemento HTML.
valor de la propiedad
las propiedades deben tener un valor. Cada propiedad tiene valores definidos. Cada valor se ubicará a la derecha.

1.5-¿Qué es un selector en CSS?

Un selector CSS es la primera parte de una regla CSS. Es un patrón de elementos y otros términos que indican al navegador qué elementos HTML se seleccionan para aplicarles una regla que incluye los valores de las propiedades CSS. El elemento o los elementos seleccionados por el selector se denominan sujeto del selector.

1.6-Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

selector de ID: Un selector de ID comienza con un carácter #, se utiliza de la misma manera que un selector de clase. Sin embargo, un ID se puede utilizar una sola vez por documento, y a cada elemento solo se le puede aplicar un id. Puede seleccionar un elemento que tenga propiedad id y ese ID puede ir precedido de un selector de tipo que seleccionará el elemento solo si el elemento y el ID coinciden

selector de clases: El selector de clase comienza con un punto (.) y selecciona todo elemento del documento que esté afectado por esa clase.

selector universal: El selector universal se indica con un asterisco (*) y se encarga de seleccionar todos los elementos de un documento.

selector de etiquetas: selecciona todas las etiquetas de un documento.

selector descendente: Los selectores descendientes son similares a selectores hijos , pero que no requieren que la relación entre los elementos coincidentes ser estrictamente entre padres e hijos.

1.7-Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS? Guía para respuesta

  1. El navegador carga el HTML.
  2. Convierte el HTML en un DOM.
  3. El navegador buscar la mayor parte de los recursos vinculados al documento HTML.
  4. El navegador analiza el CSS y ordena en diferentes cubos las reglas según el tipo de selector, con cada tipo de selector que encuentre, calcula qué reglas deben aplicarse y a qué nodos en el DOM se les aplica el estilo según corresponda
  5. El árbol de renderización muestra la estructura con la que los nodos deben aparecer después de aplicarle las reglas.
  6. En la pantalla se muestra el aspecto visual de la página.

1.8- ¿Cuáles son las tres formas para vincular CSS con HTML?

Existen 3 formas de vincular CSS con HTML y estas son: Vinculando a un archivo de hoja de estilo separado, incrustando CSS en una etiqueta de estilo Y agregando estilos en línea a elementos HTML.
